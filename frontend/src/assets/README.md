Place the following image in this directory:

**Filename:** `avatar-sathvik.jpg`

**Specifications:**
- Format: JPG or PNG
- Recommended size: 320x320px minimum
- Aspect ratio: 1:1 (square)
- Quality: High resolution for retina displays
- Content: Professional photo with good lighting

**Usage:** Used on the About page for the developer profile section with premium styling and verification badge.
