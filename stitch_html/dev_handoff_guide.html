# PaperIQ Developer Hand-off Guide (for Kiro) ## Project Overview PaperIQ is a high-fidelity AI-powered exam intelligence platform for engineering students. The UI is built on a "Dark Premium" aesthetic, prioritizing data density, confidence metrics, and actionable revision paths. ## Tech Stack Requirements - **Framework**: React - **Styling**: Tailwind CSS - **Typography**: Space Grotesk (Headlines), Geist (Body), JetBrains Mono (Data/Metrics) - **Icons**: Lucide React or Material Symbols - **Motion**: Framer Motion (for glassmorphic reveals and timeline transitions) ## Design System Tokens (PaperIQ Dark Premium) - **Background**: `#07070d` - **Surface**: `rgba(15, 15, 15, 0.75)` with backdrop-blur (Glassmorphism) - **Primary/Accent**: `#f97316` (Orange Glow) - **Success**: `#10b981` - **Border/Outline**: `rgba(255, 255, 255, 0.1)` ## Core Screen Architecture & Routes Kiro, implement the following routes based on the screens in the Stitch canvas: ### 1.
